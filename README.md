# Sales Forecasting & Business Performance Dashboard

## Project Overview
This project focuses on analyzing retail sales data to understand sales trends, store performance, and department-level insights.

An interactive Power BI dashboard was developed to help businesses monitor sales performance and support data-driven decision-making.

---

## Objectives
- Analyze weekly sales performance
- Identify high-performing stores and departments
- Visualize sales trends
- Provide business insights through dashboards

---

## Dataset
The dataset includes retail store sales information.

Key variables include:

- Store
- Department
- Weekly Sales
- Date
- Holiday Flag
- Store Type

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Power BI

---

## Data Processing
The dataset was cleaned and processed using Python.

Steps included:

- Handling missing values
- Data transformation
- Exporting cleaned data for dashboard development

---

## Dashboard Features

### KPI Cards
- Total Sales
- Average Weekly Sales
- Total Stores

### Visualizations
- Weekly Sales Trend
- Sales by Store
- Sales by Department
- Store Type Analysis
- Holiday vs Non-Holiday Sales

---

## Insights
- Certain stores consistently generate higher revenue
- Holiday seasons significantly impact sales
- Some departments outperform others in weekly sales

---

## Business Value
The dashboard helps businesses:

- Track sales performance
- Identify growth opportunities
- Improve strategic decision-making
